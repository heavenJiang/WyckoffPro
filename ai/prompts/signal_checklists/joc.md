# JOC（跳出冰点/突破Creek）信号证伪检查清单（共5条）

请逐条检验以下条件，判断这个 JOC 信号是否真实：

① **放量突破**：突破日成交量是否为近期最大成交量之一（85百分位以上）？
   - 缩量突破Creek → 缺乏需求支撑，极大概率是假突破

② **持续性**：突破后1-3日是否立刻回落到Creek下方？
   - 突破后即时回落 → 这很可能是向上试探（UT），而非有效JOC

③ **回测缩量确认**：突破后第一次回测Creek时，成交量是否明显缩量？
   - 回测放量 → 供应仍然存在，突破可靠性存疑

④ **九大检验支撑**：在JOC发生时，九大买入检验通过了几项？
   - 通过不足5项 → 基础条件不足，JOC可信度偏低

⑤ **TR充分性**：TR持续时间和横向宽度是否足够？
   - 时间不足20日 → 因（吸筹时间）不够充分，可能无法支撑有效JOC
   - 横向幅度太窄 → 横向计数目标有限，上涨空间受限
